# 10X CRM

Customer Relationship Management (CRM) System built with HTML, CSS and JavaScript.

## Features

- User Registration
- User Login
- Protected Dashboard
- Client Management (Search, Filter, Sort, Add, Delete, and Note Log)
- LocalStorage Persistence
- Dark / Light Theme
- Profile Management
- Responsive Design

## Technologies

- HTML5
- CSS3 (with Custom Variables for Theme Toggle)
- JavaScript (ES6+, Async/Await, Array Methods)
- LocalStorage
- DummyJSON API (for mocking initial database state)
